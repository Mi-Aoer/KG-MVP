# 接口说明简表

当前版本只覆盖“一期知识图谱构建闭环”。

统一返回格式：

```json
{
  "code": 0,
  "message": "ok",
  "data": {}
}
```

## 配置与项目

| Method | Path | 说明 |
| --- | --- | --- |
| POST | `/api/configs` | 新建抽取配置 |
| GET | `/api/configs` | 获取配置列表 |
| GET | `/api/configs/{id}` | 获取配置详情 |
| PUT | `/api/configs/{id}` | 更新抽取配置 |
| DELETE | `/api/configs/{id}` | 删除抽取配置 |
| POST | `/api/projects` | 新建项目 |
| GET | `/api/projects` | 项目列表 |
| GET | `/api/projects/{id}` | 项目详情 |
| PUT | `/api/projects/{id}` | 更新项目 |
| DELETE | `/api/projects/{id}` | 删除项目并联动删图 |

## 批次与抽取

| Method | Path | 说明 |
| --- | --- | --- |
| POST | `/api/projects/{project_id}/batches/upload` | 上传 txt 并创建批次 |
| GET | `/api/projects/{project_id}/batches` | 项目批次列表 |
| GET | `/api/batches/{batch_id}` | 批次详情 |
| GET | `/api/batches/{batch_id}/sources` | 批次 source 列表 |
| POST | `/api/batches/{batch_id}/extract` | 发起批量抽取 |
| GET | `/api/batches/{batch_id}/progress` | 轮询抽取进度 |
| POST | `/api/batches/{batch_id}/retry-failed` | 重试失败项 |
| POST | `/api/sources/{source_id}/retry` | 重试单条 source |

## 原始响应与三元组

| Method | Path | 说明 |
| --- | --- | --- |
| GET | `/api/sources/{source_id}` | 获取单条 source 详情 |
| PUT | `/api/sources/{source_id}/raw-response` | 修改原始响应 |
| POST | `/api/sources/{source_id}/reparse` | 重新解析当前 raw_response |
| GET | `/api/sources/{source_id}/triples` | 获取三元组列表 |
| POST | `/api/sources/{source_id}/triples` | 手工新增三元组 |
| PUT | `/api/triples/{triple_id}` | 编辑三元组 |
| DELETE | `/api/triples/{triple_id}` | 删除三元组 |

## Schema

| Method | Path | 说明 |
| --- | --- | --- |
| POST | `/api/projects/{project_id}/schema/refresh` | 从有效三元组补充 schema |
| GET | `/api/projects/{project_id}/schema` | 获取项目 schema |
| POST | `/api/projects/{project_id}/entity-types` | 新增实体类型 |
| PUT | `/api/entity-types/{id}` | 重命名实体类型 |
| DELETE | `/api/entity-types/{id}` | 删除实体类型 |
| POST | `/api/projects/{project_id}/relation-types` | 新增关系类型 |
| PUT | `/api/relation-types/{id}` | 重命名关系类型 |
| DELETE | `/api/relation-types/{id}` | 删除关系类型 |

## 图谱

| Method | Path | 说明 |
| --- | --- | --- |
| POST | `/api/projects/{project_id}/graph/init` | 初始化图谱 |
| POST | `/api/projects/{project_id}/graph/import` | 导入当前项目有效三元组 |
| POST | `/api/projects/{project_id}/graph/rebuild` | 清空当前项目图数据并重建 |
| GET | `/api/projects/{project_id}/graph/import-logs` | 导入日志列表 |

## 当前不包含

以下内容不属于当前一期接口范围：

- 问答接口
- Cypher 生成接口
- Cypher 校验接口
- 问答日志接口
